
## Типовые

Тип для недвижки:
- 0 - Новостройка
- 1 - Котеджный поселок
- 2 - Вилла
- 3 - Шалле
- 4 - Комерч.

Типы жилья:
- 0 - Вторичная
- 1 - Новостройка
- 2 - Пентхаус

Типы ремонта: 
- 0 - Без ремонта
- 1 - С ремонтом
- 2 - Дизайнерский

Параметры для поиска: 

- Поиск = search
- Тип жилья = type_jk
- Продажа = sale - (1 true \ 0 false)
- Аренда = rent - (1 true \ 0 false)
- Сутки = daily - (1 true \ 0 false)
- Тип участка = type_plan
- Комнат = rooms
- от = minPrice
- до = maxPrice
- город = City
